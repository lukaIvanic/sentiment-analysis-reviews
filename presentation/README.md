# Presentation

Final presentation deliverable for the sentiment-analysis project.

- `analiza_sentimenta_recenzija.pptx`: editable PowerPoint deck for the course presentation.

The deck summarizes the assignment, data, TF-IDF pipeline, required classifier
results, final ensemble, optional transformer extension, and submission
contents.

Several slides reuse labeled generated infographics from `figures/generated/`.
The final PPTX render was checked as a 12-slide contact sheet, and the layout
checker reported 0 errors and 0 warnings.
